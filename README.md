# MulmoChat

This repository is set up with [Vue 3](https://vuejs.org/), [Vite](https://vitejs.dev/), and [Tailwind CSS](https://tailwindcss.com/).

## Getting Started

Install dependencies:

```sh
yarn install
```

Create .env file with following API ke

```
OPENAI_API_KEY=...
GEMINI_API_KEY=...
```

Start a development server:

```sh
yarn dev
```
